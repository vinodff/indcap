## 1. Professional Color Palettes

### 1.1 Tech/AI Content (Cool, Modern, Professional)

**Use Case:** Software tutorials, AI explainers, tech reviews, developer content

| Element | Hex Code | RGB | Usage | Accessibility |
|---------|----------|-----|-------|----------------|
| **Primary Color** | `#00D9FF` | (0, 217, 255) | Main text, headlines | AAA contrast: ✓ on dark |
| **Secondary Color** | `#00FF88` | (0, 255, 136) | Accent text, emphasis | AAA contrast: ✓ on dark |
| **Tertiary Color** | `#3A86FF` | (58, 134, 255) | Info, links, supporting text | AAA contrast: ✓ on dark |
| **Background (Light)** | `#0F1419` | (15, 20, 25) | Canvas background | Minimal glare |
| **Background (Medium)** | `#1A2332` | (26, 35, 50) | Secondary background | Better contrast |
| **Background (Dark)** | `#0A0E12` | (10, 14, 18) | Safe area, borders | Maximum readability |
| **Text (Primary)** | `#FFFFFF` | (255, 255, 255) | Body text, subtitles | 21:1 contrast ratio |
| **Text (Secondary)** | `#E0E0E0` | (224, 224, 224) | Filler words, connectors | 15:1 contrast ratio |

**Gradient Combinations:**
- **Neon Cyan-to-Purple:** `linear-gradient(135deg, #00D9FF, #8338EC)`
- **Electric Blue-to-Green:** `linear-gradient(135deg, #3A86FF, #00FF88)`
- **Tech Glow:** `linear-gradient(135deg, #00D9FF 0%, #FF006E 100%)`

**Emotion-to-Color Mapping:**
- **Authority/Confidence:** `#00D9FF` (Cyan) — Tech-forward, trustworthy
- **Success/Growth:** `#00FF88` (Lime) — Positive momentum
- **Alert/Urgent:** `#FF006E` (Magenta) — High priority
- **Neutral/Connector:** `#808080` (Gray) — De-emphasis

**Best Practices:**
- Use `#00D9FF` for hero words and key metrics
- Reserve `#FF006E` for warnings or urgent calls-to-action
- Keep background ≤ 30% brightness for safe color contrast
- Never use pure white on pure black in video (anti-aliasing artifacts)

### 2.2 Font Size Hierarchy

For **1080×1920 vertical canvas** (common for Reels, TikTok, YouTube Shorts):

| Role | Size (px) | Use Case | Max Chars/Line |
|------|-----------|----------|----------------|
| **Hero/Title** | 130 | Single emphasis word | 3-4 |
| **Emphasis** | 100 | Key statements (2-3 words) | 5-7 |
| **Regular/Main** | 52 | Standard text | 8-12 |
| **Secondary** | 42 | Supporting information | 10-14 |
| **Tertiary** | 36 | Metadata, labels | 12-16 |
| **Connector** | 32 | Filler words ("the", "and") | 14-18 |
| **Caption/CTA** | 28 | Fine print, CTAs | 16-20 |

**Responsive Scaling (for mobile preview):**
```typescript
// Canvas: 1080×1920 (design size)
// Device sizes:
// - Small phone (375w): scale = 0.347 → 130px becomes 45px
// - Regular phone (414w): scale = 0.383 → 130px becomes 50px
// - Large phone (600w): scale = 0.556 → 130px becomes 72px

const calculateResponsiveFontSize = (designSize: number, deviceWidth: number) => {
  const designWidth = 1080;
  const scale = Math.min(deviceWidth / designWidth, 1.0);
  return Math.round(designSize * scale);
};
```

### 2.4 Letter Spacing & Line Height

**Letter Spacing (Tracking):**
```typescript
const letterSpacingByRole = {
  hero: {
    uppercase: 2.0,        // MONTSERRAT in caps needs breathing room
    regular: 0.5,          // Regular casing
  },
  emphasis: {
    uppercase: 1.5,
    regular: 0.3,
  },
  regular: {
    default: 0.5,
  },
  connector: {
    default: 0.0,          // No tracking for tiny text
  },
};

// Canvas implementation (in pixels):
// Letter-spacing = (tracking / 1000) * font-size
// Example: 2.0 tracking × 130px font = (2.0 / 1000) × 130 = 0.26px ≈ skip
// Practical: use 2px for visual separation

const canvasLetterSpacing = {
  hero: '2px',
  emphasis: '1px',
  regular: '0.5px',
  connector: '0px',
};
```

**Line Height (for multi-line text):**
```typescript
const lineHeightByUse = {
  singleLine: 1.0,           // Hero text (130px)
  twoLines: 1.1,             // Emphasis (100px)
  regular: 1.2,              // Body (52px)
  multiLine: 1.3,            // Longer passages
};

// Canvas implementation (vertical spacing between lines):
const canvasLineHeight = {
  singleLine: '130px',       // No spacing needed
  twoLines: '110px',         // 10px gap between lines
  regular: '62.4px',         // 1.2 × 52px
  multiLine: '67.6px',       // 1.3 × 52px
};
```

### 2.5 Text Animation Entry Speeds (Critical for Readability)

**Speed hierarchy by font size:**
| Font Size | Entry Duration | Hold Duration | Exit Duration | Why |
|-----------|----------------|---------------|---------------|-----|
| **130px** | 0.20s | 0.60s | 0.15s | Large text = needs less time to register |
| **100px** | 0.25s | 0.50s | 0.20s | Still large; brief hold |
| **52px** | 0.30s | 0.80s | 0.25s | Standard; longer for reading |
| **36px** | 0.35s | 1.0s | 0.30s | Smaller; needs more time |
| **32px** (connector) | 0.25s | 0.30s | 0.15s | Filler; fade in/out fast |

**Easing functions (industry standard):**
```typescript
const easingFunctions = {
  entryEasing: 'ease-out',          // Cubic-Bezier(0.25, 0.46, 0.45, 0.94)
                                    // Fast start, smooth landing
  exitEasing: 'ease-in',            // Cubic-Bezier(0.42, 0, 1, 1)
                                    // Smooth start, accelerate off
  transitionEasing: 'ease-in-out',  // Smooth both directions
  emphasisEasing: 'overshoot-ease-out', // Bouncy feel (0.34, 1.56, 0.64, 1)
};

// Canvas implementation (Bezier curves):
const canvasEasing = {
  'ease-out': (t: number) => 1 - Math.pow(1 - t, 3),      // Smooth decel
  'ease-in': (t: number) => t * t * t,                    // Smooth accel
  'ease-in-out': (t: number) => t < 0.5 
    ? 4 * t * t * t 
    : 1 - Math.pow(-2 * t + 2, 3) / 2,
  'overshoot': (t: number) => {
    const c1 = 1.70158;
    const c3 = c1 + 1;
    return c3 * t * t * t - c1 * t * t;
  },
};
```

```typescript
const iconWidth = 80;

const xOffset = textX + textWidth + 20;    // 20px gap from text

const iconBelowX = textX + (textWidth / 2) - (iconWidth / 2);
```

### 3.2 Icon Color Coordination

**Rules:**
1. **Match icon color to text color** (for cohesion)
2. **Use accent color** if text is neutral (draw attention)
3. **High contrast:** Icon color ≠ background color
4. **Avoid colorful icons** on dynamic backgrounds

**Examples:**
```typescript
// Tech palette: #00D9FF primary text
const techIcon = {
  color: '#00D9FF',          // Match primary
  glowIntensity: 0.5,        // Subtle glow
  rotation: 0,               // Static
  animation: 'float',        // Gentle bounce
};

// Marketing palette: #FF006E primary
const marketingIcon = {
  color: '#FF006E',          // Match primary
  glowIntensity: 0.8,        // Prominent glow
  rotation: 0,
  animation: 'scale-pulse',  // Attention-grabbing
};

// Lifestyle palette: #D4A574 primary
const lifestyleIcon = {
  color: '#A0D8A8',          // Softer accent
  glowIntensity: 0.0,        // No glow
  rotation: 0,
  animation: 'gentle-float', // Calm, slow
};
```

## 4. Animation Standards

### 4.1 Entry Animations (Word Appear)

**Timing constraints:** 200-400ms (entry) + 400-1500ms (hold) + 150-300ms (exit)

#### **Animation 1: Fade-In**
```typescript
{
  name: 'fade-in',
  duration: 0.4,           // 400ms
  easing: 'ease-out',
  effect: 'Smooth alpha transition from 0 to 1',
  intensity: 1,            // Low energy
  emotion: 'neutral',
  code: `
    canvas.globalAlpha = 0;
    for (let t = 0; t <= 1; t += 0.016) {
      canvas.globalAlpha = easeOut(t);
      drawText(text, x, y);
    }
  `,
  bestFor: ['neutral', 'connector', 'narrative'],
  avoidAfter: ['fade-in', 'fade-in'],  // Max 2 in a row
}
```

#### **Animation 2: Slide-Up**
```typescript
{
  name: 'slide-up',
  duration: 0.35,
  easing: 'ease-out',
  effect: 'Text enters from bottom (y + 100) to final position',
  intensity: 2,
  emotion: 'positive',
  code: `
    const startY = y + 100;
    for (let t = 0; t <= 1; t += 0.016) {
      const currentY = startY - (100 * easeOut(t));
      drawText(text, x, currentY);
    }
  `,
  bestFor: ['action', 'positive', 'momentum'],
  avoidAfter: ['slide-up', 'slide-down'],  // Not 2 same direction
}
```

#### **Animation 3: Scale-Up (Pop)**
```typescript
{
  name: 'scale-up',
  duration: 0.35,
  easing: 'ease-out',
  effect: 'Text grows from 0.3x to 1.0x',
  intensity: 2,
  scaleStart: 0.3,
  scaleEnd: 1.0,
  emotion: 'emphasis',
  code: `
    for (let t = 0; t <= 1; t += 0.016) {
      const scale = 0.3 + (0.7 * easeOut(t));
      ctx.save();
      ctx.translate(x + (textWidth * 0.5), y);
      ctx.scale(scale, scale);
      ctx.translate(-(textWidth * 0.5), 0);
      drawText(text, x, y);
      ctx.restore();
    }
  `,
  bestFor: ['emphasis', 'hero-word', 'statistics'],
}
```

#### **Animation 4: Typewriter**
```typescript
{
  name: 'typewriter',
  duration: 0.5,           // Character-by-character = longer
  easing: 'linear',
  effect: 'Sequential character reveal (left to right)',
  intensity: 2,
  charsPerSecond: 20,      // Speed of character reveal
  emotion: 'technical',
  code: `
    const charDuration = duration / text.length;
    for (let char = 0; char < text.length; char++) {
      const displayText = text.substring(0, char + 1);
      const charTime = (char + 1) * charDuration;
      if (currentTime >= charTime) {
        drawText(displayText, x, y);
      }
    }
  `,
  bestFor: ['tech', 'list-items', 'statistics'],
}
```

#### **Animation 5: Bounce-In**
```typescript
{
  name: 'bounce-in',
  duration: 0.4,
  easing: 'overshoot-ease-out',
  effect: 'Text bounces from -50px with overshoot',
  intensity: 3,
  bounceAmount: 50,        // Pixels of bounce
  emotion: 'joy',
  bounceCurve: 'cubic-bezier(0.34, 1.56, 0.64, 1)',
  code: `
    const bounceEasing = (t) => {
      const c1 = 1.70158;
      const c3 = c1 + 1;
      return c3 * t * t * t - c1 * t * t;
    };
    for (let t = 0; t <= 1; t += 0.016) {
      const offset = -50 + (50 * bounceEasing(t));
      drawText(text, x, y + offset);
    }
  `,
  bestFor: ['humor', 'joy', 'celebration'],
}
```

### 4.3 Emphasis Animations (Mid-Screen)

**For words already on screen that need emphasis:**

#### **Glow-Pulse**
```typescript
{
  name: 'glow-pulse',
  duration: 0.3,
  repeatCount: 1,
  intensity: 2,
  glowColor: '#FFD700',
  code: `
    for (let t = 0; t <= 1; t += 0.016) {
      const glowIntensity = Math.sin(t * Math.PI) * 20;  // 0→20→0
      ctx.shadowColor = glowColor;
      ctx.shadowBlur = glowIntensity;
```

#### **Color-Flash**
```typescript
{
  name: 'color-flash',
  duration: 0.25,
  color1: '#FFFFFF',       // Original color
  color2: '#FFD700',       // Flash color
  code: `
    for (let t = 0; t <= 1; t += 0.016) {
      // Triangular wave: white→gold→white
      let colorT = t < 0.5 ? t * 2 : (1 - t) * 2;
      const r = lerp(color1.r, color2.r, colorT);
      const g = lerp(color1.g, color2.g, colorT);
      const b = lerp(color1.b, color2.b, colorT);
      ctx.fillStyle = \`rgb(\${r}, \${g}, \${b})\`;
      drawText(text, x, y);
    }
  `,
  bestFor: ['urgent', 'highlight', 'correction'],
}
```

#### **Shake**

```typescript
  duration: 0.25,
  intensity: 3,
  shakeAmount: 8,          // Pixels of lateral movement
  frequency: 10,           // Hz
  code: `
    for (let t = 0; t <= 1; t += 0.016) {
      const shakeX = Math.sin(t * frequency * Math.PI * 2) * shakeAmount;
```

### 4.4 Exit Animations (Word Disappear)

**Timing:** 150-300ms (inverse of entry speed)

Standard exit animations:
- **Fade-Out:** Reverse of fade-in (alpha 1→0)
- **Slide-Down:** Moves down 50px while fading
- **Scale-Down:** Shrinks to 0.3x while fading
- **Blur-Out:** Increases blur while fading

**Rule:** Exit should be **faster than entry** (visual psychology: exit is less impactful)

---

### 4.5 Professional Easing Curves

**Never use `linear` for entries** (feels robotic):

```typescript
// Recommended easing functions for animation
const easingLibrary = {
  // Entry animations (starts fast, ends slow)
  'ease-out': {
    bezier: 'cubic-bezier(0.25, 0.46, 0.45, 0.94)',
    formula: (t) => 1 - Math.pow(1 - t, 3),
    feel: 'smooth landing',
    use: 'fade, slide, scale entries',
  },
  
  // Emphasis animations (organic bounce)
  'overshoot-ease-out': {
    bezier: 'cubic-bezier(0.34, 1.56, 0.64, 1)',
    formula: (t) => 1 + 2.704 * Math.pow(t - 1, 3) + 1.604 * Math.pow(t - 1, 2),
    feel: 'bouncy, playful',
    use: 'bounce-in, pop-up, joy animations',
  },
  
  // Exit animations (starts slow, ends fast)
  'ease-in': {
    bezier: 'cubic-bezier(0.42, 0, 1, 1)',
    formula: (t) => t * t * t,
    feel: 'smooth acceleration',
    use: 'fade-out, scale-down, exits',
  },
  
  // Transitions (smooth both ways)
  'ease-in-out': {
    bezier: 'cubic-bezier(0.42, 0, 0.58, 1)',
    formula: (t) => t < 0.5 ? 4 * t * t * t : 1 - Math.pow(-2 * t + 2, 3) / 2,
    feel: 'cinematic',
    use: 'position shifts, color changes',
  },
};
```

**Avoid these (feel bad in video):**
- ❌ `linear` (robotic)
- ❌ `ease-in-out` for entries (sluggish start)
- ❌ `cubic-bezier(1, 1, 1, 1)` (unsupported)
